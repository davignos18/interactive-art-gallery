# Interactive Art Gallery (Unity)

### Repository
[Interactive Art Gallery](https://github.com/davignos18/interactive-art-gallery)

### Overview
Progetto di una mostra virtuale interattiva realizzata con Unity.  
Permette di navigare tra immagini in una galleria digitale con un’interfaccia semplice e intuitiva.  
L’obiettivo è creare un’esperienza immersiva per visualizzare opere d’arte in modo digitale.

### Tecnologie
- Unity  
- C#  
- Unity UI System  
- Scene Management  

### Features
- Navigazione interattiva tra le immagini della galleria  
- UI pulita e intuitiva
- Transizioni fluide tra le scene  
- Sistema di interazione con le opere tramite musiche

### How to Run
Aprire il progetto in Unity:
-Apri Unity Hub
-Clicca su Add project e seleziona la cartella del progetto
-Apri la scena principale
-Premi Play per avviare la galleria interattiva
